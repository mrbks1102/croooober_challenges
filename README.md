## 学んだ点
・繰り返し構文の速度の違い、用途を再認識することができた。  
 - whileは条件式によってループ制御、ループ回数は不定  
 - forはオブジェクトの要素を取り出してループ。ループ回数はオブジェクトに含まれる要素数  
 - untilはfalseの間に繰り返す  
 ```rb
 #時間が一致するまでは繰り返す
 until callTime == nowTime
        nowTime = Time.now.strftime('%s%L').to_i.to_s
 end
 ```
